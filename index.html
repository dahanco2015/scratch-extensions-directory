Scratch Extensions Directory

  • Click here to learn more about Scratch extensions and ScratchX.
  • Click here to submit your extension in an issue. Include the URL of your extension file (preferably hosted on GitHub Pages), and the URLs of docs or demos if you have them. If you want me to host it for you attach the file to the issue. Also include the name you want to be listed as (if you don't I'll use your username) and include a description (if you don't I will attempt to summarize what your extension does).
  • Click here to submit your extension if you hate GitHub (like I used to).
